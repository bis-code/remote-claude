remote-claude

One script to set up Claude Code on any fresh server. github.com/bis-code/remote-claude

What Gets Installed

🌐
Tailscale
Private network — connect from anywhere without exposing ports
⚡
Node.js 22
Runtime required by Claude Code
🤖
Claude Code
Anthropic's CLI for Claude — code from anywhere

Setup Flow (13 Steps)

#1 auto Detect OS
Checks for Ubuntu/Debian, verifies root permissions
#2 auto Create Non-Root User
Prompts for username, creates user with passwordless sudo. Rejects root and invalid names.
#3 auto Generate SSH Keypair
Generates ed25519 key with optional passphrase. Adds public key to authorized_keys.
#4 manual Save Keys to Phone
Displays private + public key. Copy private key into Termius (Keychain > + > Key > paste).
#5 auto Install Tailscale
Downloads and installs via official script. Skips if already present.
#6 manual Authenticate Tailscale
Run sudo tailscale up — opens a URL to sign in to your Tailscale account.
#7 manual Save Tailscale IP + Setup Phone
Displays your Tailscale IP. Install Tailscale on phone, sign in with same account.
#8 manual Verify SSH Connection
Test your connection from Termius before SSH gets hardened. If it works, continue. If not, Ctrl+C to abort safely.
#9 auto Harden SSH
Backs up config, disables password auth + root login, restarts sshd. Idempotent.
#10 auto Install Node.js 22
Adds nodesource repo, installs Node.js 22 LTS. Skips if already present.
#11 auto Install Claude Code
npm install -g @anthropic-ai/claude-code — skips if already present.
#12 manual Authenticate Claude
Run claude, type /login, open URL in browser, paste code back.
#13 done Setup Complete!
Shows connection command: ssh user@tailscale-ip then claude
